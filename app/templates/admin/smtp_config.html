What is SMTP Verification?

SMTP ping verification connects to the mail server to verify if an email address exists without sending an email.

Configuration Tips:
  • Use port 25 for standard SMTP
  • Use port 587 with TLS for secure SMTP
  • Use port 465 with SSL for encrypted SMTP
  • Some mail servers may require authentication
  • Test your configuration before saving
Guest Permissions:

Guest users need explicit permission to use SMTP verification. Grant permission in User Management.
